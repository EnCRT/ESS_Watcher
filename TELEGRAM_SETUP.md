# Инструкция по настройке Telegram для ESS Watcher

Эта инструкция поможет вам создать Telegram-бота, получить чат и узнать chatId. Всё очень просто, даже если вы делаете это впервые.

---

## Шаг 1. Создаём бота

Бот — это ваш «почтальон», который будет отправлять уведомления от устройства вам в Telegram.

1. Откройте Telegram на телефоне или компьютере.
2. В строке поиска введите **@BotFather** (официальный бот для создания других ботов).
3. Выберите его из результатов (у него синяя галочка верификации).
4. Нажмите кнопку **START** (или напишите `/start`).
5. Отправьте команду:

   ```
   /newbot
   ```

6. BotFather спросит **имя бота**. Введите любое имя, например:

   ```
   Мой ESS Watcher
   ```

7. Затем спросит **username** (логин) бота. Он должен заканчиваться на `bot`, например:

   ```
   MyESSWatcherBot
   ```

8. Готово! BotFather пришлёт сообщение с **токеном** — длинной строкой вида:

   ```
   8786571366:AAGJl57mE6-AkcRdPPGWzh3GFHJY6MB6G9Q
   ```

   **Скопируйте этот токен и сохраните!** Он понадобится для настройки устройства.

---

## Шаг 2. Создаём чат с ботом

Чтобы бот мог отправлять вам сообщения, нужно создать чат.

1. В поиске Telegram введите **username вашего бота** (тот самый, который вы придумали на Шаге 1, например `@MyESSWatcherBot`).
2. Откройте чат с ботом.
3. Нажмите кнопку **START** (или напишите `/start`).

> **Этого достаточно.** Бот теперь «знает» вас и сможет отправлять уведомления.

> 💡 **Хотите, чтобы уведомления получали несколько человек?**  
> Создайте **группу** в Telegram, добавьте туда бота и всех нужных людей. Затем узнайте chatId группы (см. Шаг 3).

---

## Шаг 3. Узнаём Chat ID (самый простой способ)

Chat ID — это уникальный номер вашего чата с ботом. Есть три простых способа его узнать.

### Способ A: через @getidsbot (рекомендуемый — самый лёгкий)

1. Найдите в Telegram бота **@getidsbot**.
2. Нажмите START.
3. Бот сразу покажет ваш **личный** User ID.
4. Теперь **перешлите** любое сообщение из вашего чата с ботом (из Шага 2) в @getidsbot.
5. Бот покажет Chat ID. Обычно это отрицательное число (например, `-1002943931073`) — скопируйте его полностью, включая минус.

### Способ B: через ссылку в браузере

1. Откройте браузер (на телефоне или компьютере).
2. В адресной строке введите ссылку, заменив `ВАШ_ТОКЕН` на токен, полученный от BotFather:

   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```

   Пример:

   ```
   https://api.telegram.org/bot8786571366:AAGJl57mE6-AkcRdPPGWzh3GFHJY6MB6G9Q/getUpdates
   ```

3. Вы увидите текст с фигурными скобками (JSON). Найдите в нём `"chat":{"id":` — число после этого и есть ваш Chat ID.
4. Скопируйте его.

### Способ C: через @RawDataBot

1. Найдите бота **@RawDataBot**.
2. Нажмите START.
3. Перешлите любое сообщение из чата с вашим ботом в @RawDataBot.
4. В ответе найдите строку `"chat":{"id":` — это ваш Chat ID.

## Шаг 4 (опционально). Если используете группы с темами (топики)

Если вы создали супергруппу с **темами** (топиками), и хотите чтобы бот писал в конкретную тему, нужен `message_thread_id`.

1. Найдите в теме номер. Он виден в ссылке-приглашении или в адресной строке веб-версии Telegram.
2. **Самый простой способ**: напишите любое сообщение от имени вашего бота в нужную тему, а затем откройте в браузере:

   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```

3. Найдите в ответе `"message_thread_id":` — это и есть ID темы. Скопируйте его.

## Итог: что нужно для настройки ESS Watcher

После выполнения инструкции у вас должно быть три значения:

| Параметр | Пример | Описание |
|----------|--------|----------|
| **Bot Token** | `8786571366:AAGJl57mE6-AkcRdPPGWzh3GFHJY6MB6G9Q` | Токен, полученный от @BotFather |
| **Chat ID** | `-1002943931073` | ID вашего чата (личного или группы) |
| **Thread ID** | `143` | ID темы в супергруппе (если не используется — оставьте `null`) |

Эти данные вводятся через веб-интерфейс устройства в режиме настройки (зажать кнопку BOOT на 2 секунды → синий светодиод → подключиться к WiFi `ESS Watcher` пароль `12345678` → открыть `192.168.0.11`).

---

## Проверка работы

Чтобы убедиться, что всё настроено правильно:

1. В браузере откройте ссылку (замените токен и chat_id на свои):

   ```
   https://api.telegram.org/botВАШ_ТОКЕН/sendMessage?chat_id=ВАШ_CHAT_ID&text=Тест
   ```

2. Если вам пришло сообщение «Тест» — всё работает!
